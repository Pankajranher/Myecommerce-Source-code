## Quick Start (ZIP Download)

### 1️⃣ Extract the ZIP
- Right-click the downloaded ZIP file → **Extract All**  
- Navigate into the extracted folder.  

### 2️⃣ Folder Structure
After extracting, your folder should look like this:

> Note: `node_modules/` and `build/` are **not included** in the ZIP.  

### 3️⃣ Open Terminal
Open a terminal (Command Prompt / PowerShell) in the `myecommerce` folder.

### 4️⃣ Install Dependencies
Run:

```bash
npm install

start the app
npm start
